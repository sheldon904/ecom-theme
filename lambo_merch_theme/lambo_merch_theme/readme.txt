== Description ==

Lambo Merch is a premium WordPress theme designed specifically for e-commerce websites selling luxury automotive merchandise. The theme features a sleek, modern design with black and red accents, optimized for showcasing high-end products.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click 'Upload Theme' and Choose File, then select the theme's .zip file. Click 'Install Now'.
3. Click 'Activate' to use your new theme right away.
4. Navigate to Appearance > Customize to configure theme settings.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Lambo Merch is designed to work seamlessly with WooCommerce for e-commerce functionality.

= How do I set up the homepage as shown in the demo? =

You can create a page and assign the 'Home Page' template to it, then set that page as your static front page in Settings > Reading.

= How do I customize the theme colors? =

You can customize the theme colors through the Customizer in Appearance > Customize.
